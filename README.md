# 💰 Taxi Fare Price Prediction

> An end-to-end **regression model** to predict taxi fares using PyTorch with advanced feature engineering.

## 🎯 Problem Statement
Predict taxi fares accurately based on pickup/dropoff locations, time of day, and other features — enabling better pricing transparency for passengers and drivers.

## 🛠️ Tech Stack
- **Framework**: PyTorch
- **Data Processing**: Pandas, NumPy
- **Visualization**: Matplotlib, Seaborn

## 📊 Features Used
- Pickup & dropoff coordinates
- Distance (Haversine formula)
- Time of day / day of week
- Passenger count
